This folder holds the curated Part 1 dataset used for the pipeline.

Instructions:
- Place exactly 30 images in the `image/` subfolder. These are the raw inputs for part1.
- You can run `python scripts/run_pipeline.py` to copy a random 30 images from
  `data/sample/part1/image` into this folder, then preprocess -> segment.

See `scripts/run_pipeline.py` for options and invocation details.
